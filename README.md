# jobspy-plugin

`jobspy-plugin` packages the OpenClaw job search and application tracking helpers as a normal Python project.

## Install

```bash
pip install jobspy-plugin
```

## Use

```bash
jobspy --help
jobspy search --search-term "software engineer" --location "Austin, TX"
jobspy tracker list
```

## Contents

- `jobspy` console script
- `skills.jobspy` Python package
- packaged OpenClaw skill metadata
